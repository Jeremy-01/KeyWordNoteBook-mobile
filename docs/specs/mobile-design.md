## 一、项目概述

### 1.1 背景与目标

当前 KeyWordNoteBook（密码本管理器）是一款基于 PyQt5 的桌面端应用。为了扩展用户使用场景，计划迁移至移动端（iOS/Android），并引入账号系统和云端同步功能，实现多设备数据互通。

### 1.2 核心需求

| 需求类型 | 描述 |
|---------|------|
| 移动端支持 | 开发 iOS/Android 应用，使用 Flutter 框架 |
| 账号系统 | 支持用户注册、登录、密码找回 |
| 云端同步 | 密码本数据云端加密存储，多设备实时同步 |
| 生物识别 | 支持指纹/面容快速解锁 |

### 1.3 技术选型

| 组件 | 技术方案 | 说明 |
|-----|---------|------|
| 移动端框架 | Flutter 3.x | 跨平台 iOS/Android |
| 状态管理 | Riverpod | Flutter 状态管理 |
| 网络 | Dio | HTTP 客户端 |
| 本地存储 | SQLite + SecureStorage | 本地加密存储 |
| 加密 | Argon2 + AES-256 | 端到端加密 |
| 生物识别 | local_auth | 系统级指纹/面容认证 |

## 五、安全设计

### 5.1 密钥派生

```
用户主密码 (Master Key)
         │
         ├──► PBKDF2-HMAC-SHA256 ──► AES-256 Key
         │
         └──► PBKDF2-HMAC-SHA256 ──► HMAC-SHA256 Key
```

### 5.2 数据加密

- 密码使用 AES-256-GCM 加密存储
- HMAC-SHA256 保证数据完整性
- 敏感数据存储在 flutter_secure_storage
