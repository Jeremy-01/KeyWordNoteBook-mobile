## 🎯 阅读路径

### 新贡献者入门

```
1. 项目概览 (README.md)
   └─→ 了解项目是什么、核心特性

2. 贡献指南 (CONTRIBUTING.md)
   └─→ 了解如何参与开发、分支规范、提交规范

3. 开发指南 (docs/development.md)
   └─→ 了解如何搭建环境、运行项目

4. 架构设计 (docs/architecture.md)
   └─→ 深入理解系统设计和技术选型
```

## 📝 文档贡献

发现文档问题或有改进建议？

1. 提交 Issue 描述问题或建议
2. Fork 项目并修改文档
3. 创建 Pull Request

文档使用 Markdown 编写，遵循以下规范：
- 中文优先
- 简洁清晰
- 代码示例完整可运行
- 链接有效
